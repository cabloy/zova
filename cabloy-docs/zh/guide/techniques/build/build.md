# 构建

## 开发服务

```bash
$ npm run dev
```

- Env 环境变量

  | 名称            | 说明                                                                                          |
  | --------------- | --------------------------------------------------------------------------------------------- |
  | DEV_SERVER_HOST | 开发服务的host [Vite: server.host](https://vitejs.dev/config/server-options.html#server-host) |
  | DEV_SERVER_PORT | 开发服务的port                                                                                |

## 构建

```bash
$ npm run build
```

## 预览

```bash
$ npm run preview
```
